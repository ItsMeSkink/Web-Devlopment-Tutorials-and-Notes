Introduction to CSS

There are three types of CSS implementation:
  1. This is INLINE CSS. It refers to the Style Customization within the text syntax.

  2. This is Internal CSS. This refers to the style customization within the document by putting the <style> tag within the head tag. It is outside the text tag.

  3. This is External CSS. This means that the Style customization is done outside the HTML document with the help of an additional linked file with .css extension. If 2 or more customizations are overrding each other then the text will follow the precedence. Means the one that is written after the first customization, the text will follow that. Bu tot make that an exception, put "!important" under the text tag
